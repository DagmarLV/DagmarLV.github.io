Descripción:

El sistema StudyPower es una plataforma web de organización del tiempo que proporciona a los estudiantes universitarios una variedad de herramientas para ayudarlos a tener éxito en sus estudios. El sistema estará potenciado por LLM (Large Language Models), lo que le permite ofrecer funciones avanzadas como estrategias de priorización en la gestión de tiempo, búsqueda de información, generación de resúmenes y de contenido motivacional personalizado.
